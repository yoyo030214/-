服务条款
1. 服务说明

农业应用商家管理后台（以下简称"平台"）是为农产品商家提供的管理工具，用于管理商品、订单、客户等信息。

2. 账号注册与使用

2.1 商家需要提供真实、准确、完整的注册信息，并保证这些信息的合法性。

2.2 商家应妥善保管账号和密码，因账号或密码保管不当造成的损失由商家自行承担。

3. 商品管理

3.1 商家应确保在平台上发布的商品信息真实、准确。

3.2 商家不得发布违反国家法律法规的商品。

4. 订单与配送

4.1 商家应及时处理订单，保证产品质量。

4.2 商家应按照约定时间配送商品。

5. 违规处理

5.1 如商家违反本服务条款，平台有权暂停或终止商家账号。

5.2 因违规行为导致的法律责任由商家自行承担。

隐私政策
1. 信息收集

1.1 我们收集您的基本信息，包括用户名、商家名称、联系方式等。

1.2 我们收集您在使用平台过程中产生的交易数据、浏览记录等。

2. 信息使用

2.1 我们使用您的信息来提供、维护和改进平台服务。

2.2 我们可能会向您发送服务通知、活动信息等。

3. 信息保护

3.1 我们采取合理的安全措施保护您的信息。

3.2 我们不会将您的信息出售、出租或以其他方式提供给第三方。

4. Cookie的使用

4.1 我们使用Cookie来改善用户体验。

4.2 您可以通过浏览器设置拒绝Cookie，但这可能影响部分功能的使用。

5. 修改与更新

5.1 本隐私政策可能会不定期更新。

5.2 更新后的隐私政策将在平台上公布。
